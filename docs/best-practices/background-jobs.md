# <a name="background-jobs"></a><span data-ttu-id="b4a4a-103">后台作业</span><span class="sxs-lookup"><span data-stu-id="b4a4a-103">Background jobs</span></span>

<span data-ttu-id="b4a4a-104">许多类型的应用程序需要运行与用户界面 (UI) 无关的后台任务。</span><span class="sxs-lookup"><span data-stu-id="b4a4a-104">Many types of applications require background tasks that run independently of the user interface (UI).</span></span> <span data-ttu-id="b4a4a-105">示例包括批处理作业、密集处理型任务，以及长时间运行的过程，例如工作流。</span><span class="sxs-lookup"><span data-stu-id="b4a4a-105">Examples include batch jobs, intensive processing tasks, and long-running processes such as workflows.</span></span> <span data-ttu-id="b4a4a-106">后台作业无需用户交互就可执行 -- 应用程序可以启动作业，并继续处理来自用户的交互请求。</span><span class="sxs-lookup"><span data-stu-id="b4a4a-106">Background jobs can be executed without requiring user interaction--the application can start the job and then continue to process interactive requests from users.</span></span> <span data-ttu-id="b4a4a-107">这有助于减少应用程序 UI 上的负载，从而提高可用性，降低交互响应时间。</span><span class="sxs-lookup"><span data-stu-id="b4a4a-107">This can help to minimize the load on the application UI, which can improve availability and reduce interactive response times.</span></span>

<span data-ttu-id="b4a4a-108">例如，如果应用程序需要生成用户上传的图像缩图，可以后台作业的形式执行此操作，并在完成时会缩图保存到存储中，用户无需等待过程完成。</span><span class="sxs-lookup"><span data-stu-id="b4a4a-108">For example, if an application is required to generate thumbnails of images that are uploaded by users, it can do this as a background job and save the thumbnail to storage when it is complete--without the user needing to wait for the process to be completed.</span></span> <span data-ttu-id="b4a4a-109">同样地，下单的用户可以启动一个后台工作流来处理订单，同时，UI 可让用户继续浏览 Web 应用。</span><span class="sxs-lookup"><span data-stu-id="b4a4a-109">In the same way, a user placing an order can initiate a background workflow that processes the order, while the UI allows the user to continue browsing the web app.</span></span> <span data-ttu-id="b4a4a-110">当后台作业完成时，可以更新存储的订单数据，并将确认订单的电子邮件发送给用户。</span><span class="sxs-lookup"><span data-stu-id="b4a4a-110">When the background job is complete, it can update the stored orders data and send an email to the user that confirms the order.</span></span>

## <a name="types-of-background-jobs"></a><span data-ttu-id="b4a4a-113">后台作业的类型</span><span class="sxs-lookup"><span data-stu-id="b4a4a-113">Types of background jobs</span></span>

## <a name="triggers"></a><span data-ttu-id="b4a4a-122">触发器</span><span class="sxs-lookup"><span data-stu-id="b4a4a-122">Triggers</span></span>
