### <a name="function-app-plans"></a><span data-ttu-id="6ad30-166">函数应用计划</span><span class="sxs-lookup"><span data-stu-id="6ad30-166">Function App plans</span></span>

<span data-ttu-id="6ad30-167">Azure Functions 支持两种托管模型。</span><span class="sxs-lookup"><span data-stu-id="6ad30-167">Azure Functions supports two hosting models.</span></span> <span data-ttu-id="6ad30-168">**消耗量计划**：运行代码时自动分配计算能力。</span><span class="sxs-lookup"><span data-stu-id="6ad30-168">With the **consumption plan**, compute power is automatically allocated when your code is running.</span></span>  <span data-ttu-id="6ad30-169">**应用服务计划**：为代码分配一组 VM。</span><span class="sxs-lookup"><span data-stu-id="6ad30-169">With the **App Service** plan, a set of VMs are allocated for your code.</span></span> <span data-ttu-id="6ad30-170">应用服务计划定义 VM 数目和 VM 大小。</span><span class="sxs-lookup"><span data-stu-id="6ad30-170">The App Service plan defines the number of VMs and the VM size.</span></span>

<span data-ttu-id="6ad30-280">在参考实现中，函数会检查经过身份验证的用户是否为 `GetStatus` 应用程序角色的成员。</span><span class="sxs-lookup"><span data-stu-id="6ad30-280">In the reference implementation, the function checks whether the authenticated user is a member of the `GetStatus` application role.</span></span> <span data-ttu-id="6ad30-281">如果不是，则函数会返回“HTTP 未授权”(401) 响应。</span><span class="sxs-lookup"><span data-stu-id="6ad30-281">If not, the function returns an HTTP Unauthorized (401) response.</span></span>

```csharp
[FunctionName("GetStatusFunction")]
public static Task<IActionResult> Run(
    [HttpTrigger(AuthorizationLevel.Function, "get", "post", Route = null)] HttpRequest req,
    [CosmosDB(
        databaseName: "%COSMOSDB_DATABASE_NAME%",
        collectionName: "%COSMOSDB_DATABASE_COL%",
        ConnectionStringSetting = "COSMOSDB_CONNECTION_STRING",
        Id = "{Query.deviceId}",
        PartitionKey = "{Query.deviceId}")] dynamic deviceStatus,
    ILogger log)
{
    log.LogInformation("Processing GetStatus request.");

    return req.HandleIfAuthorizedForRoles(new[] { GetDeviceStatusRoleName },
        async () =>
        {
            string deviceId = req.Query["deviceId"];
            if (deviceId == null)
            {
                return new BadRequestObjectResult("Missing DeviceId");
            }

            return await Task.FromResult<IActionResult>(deviceStatus != null
                    ? (ActionResult)new OkObjectResult(deviceStatus)
                    : new NotFoundResult());
        },
        log);
}
```
