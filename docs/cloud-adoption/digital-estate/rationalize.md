### <a name="inventory-reduce-discovery-data-points"></a><span data-ttu-id="1edcb-142">库存：减少发现数据点</span><span class="sxs-lookup"><span data-stu-id="1edcb-142">Inventory: Reduce discovery data points</span></span>

<span data-ttu-id="1edcb-177">但是，在转移到云记帐模型时，淘汰资产可能会明显节省年度运营成本和前期迁移工作量。</span><span class="sxs-lookup"><span data-stu-id="1edcb-177">However, when moving to a cloud accounting model, retiring assets can produce significant savings in annual operating costs and up-front migration efforts.</span></span>

<span data-ttu-id="1edcb-190">这项以计划为中心的质化分析可以快速执行，将在多个迁移积压工作之间创建基准。</span><span class="sxs-lookup"><span data-stu-id="1edcb-190">This program-focused qualitative analysis can be executed quickly and will create alignment across multiple migration backlogs.</span></span>
