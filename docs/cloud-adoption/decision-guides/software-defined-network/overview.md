<span data-ttu-id="26f47-104">软件定义网络 (SDN) 是一种网络体系结构，旨在实现可通过软件集中管理、配置和修改的虚拟化网络功能。</span><span class="sxs-lookup"><span data-stu-id="26f47-104">Software Defined Networking (SDN) is a network architecture designed to allow virtualized networking functionality that can be centrally managed, configured, and modified through software.</span></span> <span data-ttu-id="26f47-105">SDN 通过物理网络基础结构提供抽象层，并支持物理路由器、防火墙和其他可在本地网络中找到的网络硬件的虚拟化等效项。</span><span class="sxs-lookup"><span data-stu-id="26f47-105">SDN provides an abstraction layer over the physical networking infrastructure, and enables the virtualized equivalent to physical routers, firewalls, and other networking hardware you would find in an on-premises network.</span></span>

<span data-ttu-id="26f47-106">SDN 允许 IT 人员使用虚拟资源配置和部署支持工作负荷需求的网络结构和功能。</span><span class="sxs-lookup"><span data-stu-id="26f47-106">SDN allows IT staff to configure and deploy network structures and capabilities that support workload needs using virtualized resources.</span></span> <span data-ttu-id="26f47-107">基于软件的部署管理的灵活性支持快速修改网络资源并能够同时支持敏捷和传统部署模型。</span><span class="sxs-lookup"><span data-stu-id="26f47-107">The flexibility of software-based deployment management enables rapid modification of networking resources and allows the ability to support both agile and traditional deployment models.</span></span> <span data-ttu-id="26f47-108">使用 SDN 技术创建的虚拟网络对在公有云平台上创建安全的网络至关重要。</span><span class="sxs-lookup"><span data-stu-id="26f47-108">Virtualized networks created with SDN technology are critical to creating secure networks on a public cloud platform.</span></span>
